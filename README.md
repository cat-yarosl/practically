# Frontend Project

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying [`src/app/page.tsx`](src/app/page.tsx). The page auto-updates as you edit the file.

## Project Structure

- `src/app/components/`: Contains the React components used in the application.
  - [`Footer.tsx`](src/app/components/Footer.tsx): The footer component.
  - [`MainContent.tsx`](src/app/components/MainContent.tsx): The main content component.
  - [`Nav.tsx`](src/app/components/Nav.tsx): The navigation bar component.
  - [`Sidebar.tsx`](src/app/components/Sidebar.tsx): The sidebar component.
- [`src/app/globals.css`](src/app/globals.css): Global CSS styles.
- [`src/app/layout.tsx`](src/app/layout.tsx): The root layout component.
- [`src/app/page.tsx`](src/app/page.tsx): The main page component.
